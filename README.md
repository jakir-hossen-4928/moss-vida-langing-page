# 🌊 MossVida — Premium Shopify Theme Experience

MossVida is a high-performance, conversion-optimized Shopify theme designed specifically for health, wellness, and supplements. It features a suite of custom-built Liquid sections with a focus on "Ocean-Deep" aesthetics, fluid responsiveness, and full merchant control.

## ✨ Key Features

### 🚀 Performance & UX
- **Zero Dependencies**: Built with Vanilla JS and CSS for lightning-fast load times.
- **Mobile-First Design**: Every section is meticulously crafted for mobile usability.
- **AJAX Integration**: Smart "Add to Cart" functionality and dynamic variant selection without page reloads.

### 🎨 Custom Sections Suite
- **MossVida — Hero**: Animated headings, glassmorphism badges, and dynamic background overlays.
- **MossVida — Product**: A custom-designed product layout featuring interactive galleries and streamlined quantity selectors.
- **MossVida — Why Sea Moss**: A unique grid section for storytelling with decorative floating badges.
- **MossVida — Benefits & FAQ**: Polished layouts with smooth accordion transitions and grid-based value props.
- **MossVida — Reviews**: Social proof section with premium card styling and star ratings.

### ⚙️ Merchant Customization
- **Full Schema Control**: Merchants can customize every detail—colors, spacing, typography (Desktop/Mobile), and layout widths—directly through the Shopify Theme Editor.
- **Dynamic Blocks**: Flexible content management using Shopify's block system for FAQs and Benefits.

---

## 🛠️ Technology Stack

- **Shopify Liquid**: Robust templating for dynamic data injection.
- **Vanilla CSS3**: Modern properties including CSS Variables, Grid, and Flexbox for high-end styling.
- **Vanilla JavaScript**: Interactive components (Galleries, Accordions, AJAX) with a focus on performance.
- **Shopify AJAX API**: For seamless cart management.

---

## 🚀 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/moss-vida.git
   ```
2. **Push to Shopify**:
   Use the Shopify CLI to push the theme to your store:
   ```bash
   shopify theme push
   ```
3. **Customize**:
   Open the **Shopify Theme Editor**, navigate to any page, and add the sections starting with the prefix `MossVida`.
